## Lesson 0-3 Classroom instructions (p.19)
### Classroom instructions
- Teacher: いってください
    - Please say it. 
- Teacher: みてください
    - Please look. 
- Teacher: かいてください
    - Please write. 
- Teacher: よんでください
    - Please read aloud. 
- Teacher: はなしてください
    - Please speak. 
- Teacher: きいてください
    - Please listen. 
- Teacher: 〇〇さんに　きいてください
    - Please ask 〇〇san.
- Teacher: もういちど　いってください
    - Please say it one more time. 
- Teacher: おおきいこえで　いってください
    - Please say it in a louder voice. 
- Student: もういちど　おねがいします
    - Could you please say it one more time. 
- Student: おおきいこえで　おねがいします
    - Could you please say it in a louder voice. 
- Student: ゆっくり　おねがいします
    - Could you please say it more slowly. 
## Lesson 0-4 Useful phrases in class (pp.19-20)
- Teacher: わかりますか。
    - Do you understand?
    - Student: はい、わかります。
        - Yes, I understand.
    - or Student: いいえ、わかりません。
        - No, I don't understand. 
- Student: とびらせんせい、しつもんが　あります。
    - Professor Tobira, I have a question.
    - Teacher: はい、[スミス]()さん。

    - What is "Pencil" in Japanese?
    - Teacher: [Pencil]() は　にほんごで　[えんぴつ]()です。
        - "Pencil" is "enpitsu" in Japanese. 
    - or Teacher: えんぴつです。
- Student: [けしゴム]()は　えいごで　なんですか。
    - What is "keshigomu" in English?
    - Teacher: [けしゴム]()は　えいごで　[Eraser]() です。
        - Student: そうですか。
            - I see.
        - or Student: わかりました。
            - I got it.

- あ

- え
    - As in "dr[e]()ss"
- お
    - As in "n[o]()rth"
### Consonant
#### K
- かきくけこ
#### S
- さしすせそ
    - し

#### T
- たちつてと

        - As in "star", not "tar".
    - ち
        - As in "cheese"
    - つ
        - As in "I[t's oo]()zing," but with unrounded lips

## Module 4-1: Basic Intonation Patterns – Word Level
- To learn about intonation: ( http://enunciate.arts.ubc.ca/japanese/intonation/introduction-to-intonation/ ).
- To learn pitch patterns in Japanese: ( http://enunciate.arts.ubc.ca/japanese/intonation/pitch-word-level/ ).
- The first and second mora should have different pitches.  
  If the first mora is HIGH, then the second mora is LOW;  
  if the first mora is LOW, then the second mora is HIGH.  
- Within a word/phrase, once the pitch goes down it never goes back up.  
  The sentence final particles [ね、よ、か](), however, are exceptions to this rule.
